# Checklist

- [ ] 项目骨架与依赖：`requirements.txt`、`config.py`、`run.py`、`app/__init__.py` 已创建并配置正确
- [ ] 数据模型：`User`、`Category`、`Novel`、`Chapter`、`ReadingProgress`、`Bookmark`、`ChapterRule` 均已定义且字段完整
- [ ] 登录鉴权：根路径 `/` 重定向到登录页；除 `/login` 与静态资源外所有路由需登录
- [ ] 登录页：正确凭证登录成功跳转 `/dashboard`；错误凭证提示错误信息
- [ ] 登出：登出后清除 session 并重定向到 `/login`
- [ ] 首次启动：自动创建默认用户账号与内置默认章节拆分规则
- [ ] 分类管理：分类的创建、查看、编辑、删除功能可用；删除分类时关联小说 category_id 置空
- [ ] 规则管理列表：`/rules` 展示名称、正则、描述、启用状态、操作按钮
- [ ] 规则增删改：新增、编辑、删除规则可用
- [ ] 规则启用/禁用：禁用后不再出现在导入页规则下拉框
- [ ] 恢复默认规则：可补齐缺失的内置默认规则，不影响自定义规则
- [ ] TXT 导入：上传 UTF-8/GBK TXT 文件能正确识别编码无乱码
- [ ] 规则导入：选择启用规则后能按该规则正则拆分章节
- [ ] 自定义正则：导入页"自定义正则"输入框覆盖所选规则并正确拆分章节
- [ ] 序章处理：第一章标题前的引言内容归入"序章"
- [ ] 小说列表：支持按分类筛选展示
- [ ] 小说详情页：展示章节目录、阅读进度、操作按钮
- [ ] 阅读页：显示章节正文，上一章/下一章切换正常
- [ ] 阅读进度：阅读后自动记录章节与滚动位置，重开定位到上次位置
- [ ] 书签：添加、查看、跳转、删除书签功能正常
- [ ] 在线编辑：修改章节标题与正文后阅读页立即生效
- [ ] 导出 TXT：下载文件名为 `<小说标题>.txt`，章节按序拼接、空行分隔
- [ ] 全文搜索：输入关键词返回匹配章节列表，含小说标题、章节标题、摘要、跳转链接
- [ ] Flask-Admin 后台：`/admin` 需登录可访问，可对全部模型（含 ChapterRule）进行增删改查
- [ ] 响应式布局：电脑（≥992px）与手机（<992px）视口下页面布局与操作正常
- [ ] 静态资源：Layui 已正确引入并在模板中生效
- [ ] 安全性：密码使用 hash 存储（非明文）
- [ ] 可运行性：`python run.py` 能正常启动并访问
