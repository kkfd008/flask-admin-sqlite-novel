# Checklist

- [ ] 项目骨架与依赖：`requirements.txt`、`config.py`、`run.py`、`app/__init__.py` 已创建并配置正确
- [ ] 数据模型：`User`、`Category`、`Novel`、`Chapter`、`ReadingProgress`、`Bookmark`、`ChapterRule`、`NovelChapterRule`、`Tag`、`Favorite`、`Rating` 均已定义且字段完整
- [ ] 登录鉴权：根路径 `/` 重定向到登录页；除 `/login` 与静态资源外所有路由需登录
- [ ] 登录页：正确凭证登录成功跳转 `/dashboard`；错误凭证提示错误信息
- [ ] 登出：登出后清除 session 并重定向到 `/login`
- [ ] 首次启动：自动创建默认用户账号与 8 大类内置章节拆分规则
- [ ] 分类管理：分类的创建、查看、编辑、删除功能可用；删除分类时关联小说 category_id 置空
- [ ] 标签管理：标签的创建、查看、编辑、删除功能可用；删除标签时清除小说关联，不删除小说
- [ ] 标签-小说关联：小说详情页可勾选/取消勾选多个标签，列表页可按标签筛选小说
- [ ] 收藏功能：小说详情页可收藏/取消收藏，收藏列表页展示所有收藏小说
- [ ] 评分功能：可为小说打 1-5 星评分（可附评价），支持修改评分；详情页与列表页显示平均评分与评分人数
- [ ] 规则管理列表：`/rules` 展示分类、名称、正则、描述、启用状态、操作按钮
- [ ] 规则增删改：新增、编辑、删除规则可用
- [ ] 规则启用/禁用：禁用后不再出现在导入页规则下拉框
- [ ] 恢复默认规则：可补齐缺失的内置默认规则，不影响自定义规则
- [ ] 书籍章节规则：每本书可查看、编辑、删除独立的章节拆分规则（长期保存）
- [ ] 导入保存规则：导入 Step 2 填入自定义正则并勾选"保存为本书规则"后，规则长期保存
- [ ] 导入 Step 1：上传文件后自动检测编码并转换为 UTF-8，标题默认取文件名（不含扩展名），可进入下一步
- [ ] 导入 Step 2：可选择章节拆分规则或填入自定义正则，每本书独立选择
- [ ] 导入 Step 3：显示匹配出的章节标题列表（含标题、字数、内容预览），可删除章节并重新生成
- [ ] 导入 Step 4：小说标题默认为文件名，作者可为空，确认导入后章节按修改后的列表正确保存
- [ ] 导入向导：4 步之间可前进/后退，中间状态正确保存
- [ ] 序章处理：第一章标题前的引言内容归入"序章"
- [ ] 小说列表：支持按分类/标签筛选展示，显示评分与收藏状态
- [ ] 小说详情页：展示标签、章节目录、阅读进度、评分、收藏按钮、操作按钮
- [ ] 阅读页：显示章节正文，上一章/下一章切换正常
- [ ] 阅读进度：阅读后自动记录章节与滚动位置，重开定位到上次位置
- [ ] 书签：添加、查看、跳转、删除书签功能正常
- [ ] 在线编辑：修改章节标题与正文后阅读页立即生效
- [ ] 导出 TXT：下载文件名为 `<小说标题>.txt`，章节按序拼接、空行分隔
- [ ] 全文搜索：输入关键词返回匹配章节列表，含小说标题、章节标题、摘要、跳转链接
- [ ] Flask-Admin 后台：`/admin` 需登录可访问，可对全部模型（含 ChapterRule、Tag、Favorite、Rating）进行增删改查
- [ ] 响应式布局：电脑（≥992px）与手机（<992px）视口下页面布局与操作正常
- [ ] 主题切换：右上角固定位置，白天/黑夜配色正确，未登录也可切换，切换后刷新页面保持主题（localStorage）
- [ ] 静态资源：Layui 已正确引入并在模板中生效
- [ ] 安全性：密码使用 hash 存储（非明文）
- [ ] 可运行性：`python run.py` 能正常启动并访问
